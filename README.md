# 📋 TaskFlow - Modern To-Do List PWA

TaskFlow is a modern, installable Progressive Web App (PWA) designed to help users organize their daily tasks, stay productive, and never miss important deadlines. Built with **React**, **Vite**, and **Tailwind CSS**, the application provides a fast, responsive, and mobile-first experience with offline support and local notifications.

Whether you're managing personal goals, work assignments, or study schedules, TaskFlow offers an intuitive interface to keep your day organized.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/todo-pwa.git
```

Navigate into the project

```bash
cd todo-pwa
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

---
